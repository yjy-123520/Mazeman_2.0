# 迷宫游戏 MazeGame

基于Qt6和C++开发的迷宫游戏，支持键盘控制玩家移动、迷宫编辑和计时排名功能。

---

## 📖 项目报告

### 一、创作思路

#### 1.1 灵感来源
这个项目的灵感来自小时候在4399玩的迷宫游戏，希望自己实现一个可以自己编辑的迷宫游戏给别人玩。选择Qt6是因为想学习图形界面开发，C++则是比较熟悉的语言。

#### 1.2 设计理念
- **操作简单**：只用方向键或WASD就能控制移动
- **玩法保证**：玩家手动生成迷宫，保证pvp对抗性
- **可扩展**：代码模块化，方便以后添加新功能

#### 1.3 技术选型
- 编程语言：C++17
- 开发框架：Qt 6.10
- 构建工具：CMake

---

### 二、功能特性

- 键盘控制玩家移动（方向键 / WASD）
- 到达终点胜利检测与提示
- 迷宫编辑器（点击编辑墙壁）
- 多种尺寸选择（5×5、10×10、15×15）
- 实时计时功能
- QSS样式美化界面
- 过渡动画效果
